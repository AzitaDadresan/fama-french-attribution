# Fama-French Attribution Dashboard

![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

**Fama-French 3-factor + Brinson attribution for alpha decomposition and risk budgeting.**

## Features
- Fama-French 3-factor model (Market, SMB, HML)
- Brinson attribution (allocation vs selection)
- Interactive Plotly charts
- Real-time yfinance data

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

Navigate to http://localhost:8501
